1、CSS Selectors

选择符 语法 版本 描述
ID选择符(ID Selectors) #sID CSS1 以文档目录树（DOM）中作为对象的唯一标识符的ID作为选择
类型选择符(Type Selectors) E1 CSS1 以文档语言对象类型作为选择符
类选择符(Class Selectors) E1.className CSS1 在HTML中可以使用此种选择符。其效果等同于E1[class~=className]
包含选择符(Descendant Selectors) E1 E2 CSS1 选择所有被E1包含的E2。即E1.contains(E2)==true
选择符分组(Grouping) E1,E2,E3 CSS1 将同样的定义应用于多个选择符，可以将选择符以逗号分隔的方式并为组
 
通配选择符(Universal Selector) * CSS2 选定文档目录树（DOM）中的所有类型的单一对象
子对象选择符(Child Selectors) E1 > E2 CSS2 选择所有作为E1子对象的E2
相邻选择符(Adjacent Sibling Selectors) E1 + E2 CSS2 选择紧贴在对象E1之后的所有E2对象
属性选择符(Attribute Selectors) E1[attr] CSS2 选择具有attr属性的E1
属性选择符(Attribute Selectors) E1[attr=value] CSS2 选择具有attr属性且属性值等于value的E1
属性选择符(Attribute Selectors) E1[attr~=value] CSS2 选择具有attr属性且属性值为一用空格分隔的字词列表，其中一个等于value的E1
属性选择符(Attribute Selectors) E1[attr|=value] CSS2 选择具有attr属性且属性值为一用连字符分隔的字词列表，由value开始的E1
 
子串匹配的属性选择符 E[att^="val"] CSS3 匹配具有att属性、且值以val开头的E元素
子串匹配的属性选择符 E[att$="val"] CSS3 匹配具有att属性、且值以val结尾的E元素
子串匹配的属性选择符 E[att*="val"] CSS3 匹配具有att属性、且值中含有val的E元素
结构性伪类 E:root CSS3 匹配文档的根元素。在HTML中，根元素永远是HTML
结构性伪类 E:nth-child(n) CSS3 匹配父元素中的第n个子元素E
结构性伪类 E:nth-last-child(n) CSS3 匹配父元素中的倒数第n个结构子元素E
结构性伪类 E:nth-of-type(n) CSS3 匹配同类型中的第n个同级兄弟元素E
结构性伪类 E:nth-last-of-type(n) CSS3 匹配同类型中的倒数第n个同级兄弟元素E
结构性伪类 E:last-child CSS3 匹配父元素中最后一个E元素
结构性伪类 E:first-of-type CSS3 匹配同级兄弟元素中的第一个E元素
结构性伪类 E:only-child CSS3 匹配属于父元素中唯一子元素的E
结构性伪类 E:only-of-type CSS3 匹配属于同类型中唯一兄弟元素的E
结构性伪类 E:empty CSS3 匹配没有任何子元素（包括text节点）的元素E
目标伪类 :target CSS3 匹配相关URL指向的E元素
UI元素状态伪类 E:enabled CSS3 匹配所有用户界面（form表单）中处于可用状态的E元素
UI元素状态伪类 E:disabled CSS3 匹配所有用户界面（form表单）中处于不可用状态的E元素
UI元素状态伪类 E:checked CSS3 匹配所有用户界面（form表单）中处于选中状态的元素E
UI元素状态伪类 E::selection CSS3 匹配E元素中被用户选中或处于高亮状态的部分
否定伪类 E:not(s) CSS3 匹配所有不匹配简单选择符s的元素E
通用兄弟元素选择器 E ~ F CSS3 匹配E元素之后的F元素
 

2、CSS Pseudo-Classes

伪类 版本 描述
:link CSS1 设置a对象在未被访问前的样式表属性
:visited CSS1 设置a对象在其链接地址已被访问过时的样式表属性
:hover CSS1/CSS2 设置对象在其鼠标悬停时的样式表属性
:active CSS1/CSS2 设置对象在被用户激活（在鼠标点击与释放之间发生的事件）时的样式表属性
:focus CSS2 设置对象在成为输入焦点（该对象的onfocus事件发生）时的样式表属性
:first-child CSS2 设置对象（Selector1）的第一个子对象（Selector2）的样式表属性
:first CSS2 设置页面容器第一页使用的样式表属性。仅用于@page规则
:left CSS2 设置页面容器位于装订线左边的所有页面使用的样式表属性。仅用于@page规则
:right CSS2 设置页面容器位于装订线右边的所有页面使用的样式表属性。仅用于@page规则
:lang CSS2 设置对象使用特殊语言的内容样式表属性

3、CSS Pseudo-Elements

伪对象 版本 描述
:first-letter CSS2 设置对象内的第一个字符的样式表属性
:first-line CSS2 设置对象内的第一行的样式表属性
:before CSS2 用来和content属性一起使用，设置在对象前（依据对象树的逻辑结构）发生的内容
:after CSS2 用来和content属性一起使用，设置在对象后（依据对象树的逻辑结构）发生的内容
